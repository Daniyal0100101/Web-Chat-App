# Web Chat App

A real-time web chat application using Flask and Socket.io.

## Features

- Real-time messaging with Socket.io and Flask
- JavaScript frontend for a dynamic user experience
- Easy setup and deployment

## Getting Started

### Run the App Locally

1. Clone the repository:

```bash
git clone https://github.com/Daniyal0100101/Web-Chat-App.git
```

2. Navigate to the project directory:

```bash
cd Web-Chat-App
```

3. Create a virtual environment:

-   On macOS and Linux:

```bash
python3 -m venv venv
```

-   On Windows:

```bash
python -m venv venv
```

5. Activate the virtual environment:

-   On macOS and Linux:

```bash
source venv/bin/activate
```

-   On Windows:

```bash
venv\Scripts\activate
```

6. Install the dependencies:

-   On macOS and Linux:

```bash
pip3 install -r requirements.txt
```

-   On Windows:

```bash
pip install -r requirements.txt
```

7. Run the app:

-   On macOS and Linux:

```bash
python3 app.py
```

-   On Windows:

```bash
python app.py
```
